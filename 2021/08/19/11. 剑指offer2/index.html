part-01 整数

001 整数除法

注意的点：

  • 记住int基本类型的最大整数和最小整数是通过Integer.MIN_VALUE和Integer.MAX_VALUE来得到的。
  • 这里面的symbol初始值设置为0，两个数里面只要有正数就让他为1，并且把他的值转为负数。如果最后symbol的值为1,说明他俩符号相异，最后在返回的结果上加上负号。
  • 关于为什么把正数变为负数，而不是反过来，是因为复数的表示范围比正数大1，转换为负数不会溢出。

002 二进制加法

注意：

  • 字符串拼接用StringBuffer这个类，拼接用append()方法，反转用reverse()方法
  • 获取字符串特定索引处的字符用charAt()方法
  • 字符串索引的最大值比字符串长度小1
  • add 表示进位的值，求进位的值用/取商，求本位的值用%取余数（模）

003 前 n 个数字二进制中 1 的个数

注意：

  • 整型数组如果不赋初值的话，里面每个元素的默认值为0
  • j &(j-1)会消掉元素j的二进制表示中的一个1
  • 不用把j转换为二进制再运算，因为计算机本来就是计算二进制的，他自己在运算的时候会自动转换

part-02 数组

007 数组中和为 0 的三个数

注意：

  • 碰到求几个数之和的，首先想到双指针做法，并且需要将数组排好序
  • 对数组进行排序用Arrays.sort()这个方法
  • 对于本题中的三数之和为0，可以让第二个和第三个数等于第一个数的负数就行了
  • set集合里面不允许存放重复元素，所以可以把list存放在里面不用考虑重复问题
  • new ArrayList<>(set)这个构造方法可以将一个set集合转换成list集合
  • Arrays.asList()可以将若干个整数作为参数，返回的是一个list集合

008 和大于等于 target 的最短子数组

注意：

  • 关于设置res=Integer.MAX_VALUE这一语句，是为了后面用Math.min()来对满足题目的子数组进行长度比较，如果值仍为Integer.MAX_VALUE，说明没有满足题意的子数组，按照要求返回0
  • 由题目的连续子数组这一要求可以知道采用滑动窗口这一办法，值比目标值小，向右移动右指针，值大于等于目标值，向右移动左指针
  • 数组的长度等于右指针减去左指针再加一
  • 注意while循环left<=right这个条件可以取等号

010 和为 k 的子数组

不会

要用到哈希表和前缀和

前缀和：

比如有一个数组a = [a1, a2, a3, a4, a5]

前缀和就为：

sum[0] = a[0];

sum[1] = a[1] + a[0];

以此类推：

sum[4] = a[4] + a[3] + a[2] + a[1] + a[0]
